## 本地库基本操作

本地库初始化：git init

设置系统签名：git config --global user.name tom_glb    
								  git config --global goodMorning_pro@qq.com

> 项目级别优先于系统用户级别，二者都有时采用项目级别的签名

状态查看：git status

添加到暂存区：git add [filename]

将暂存区的内容提交到本地库：git commit -m "commit message" [file name]
